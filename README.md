ShopEase – E-Commerce Website

ShopEase is a simple and responsive e-commerce website developed using HTML, CSS, and JavaScript. The project focuses on building a clean user interface and implementing basic shopping features such as product listing and cart management.

This project was created as part of my learning in front-end web development and demonstrates my understanding of building interactive web applications using core web technologies.

Project Features

Responsive home page with a modern layout

Product listing page displaying multiple products

Product search functionality

Add to cart feature using JavaScript

Cart page to view and manage selected products

Option to increase or decrease product quantity

Remove items from the cart

Cart data stored using LocalStorage

Responsive design for mobile, tablet, and desktop devices

Technologies Used

HTML5 – For structuring the web pages
CSS3 – For styling and responsive layouts
JavaScript – For dynamic functionality and cart logic
LocalStorage – To store cart data in the browser
